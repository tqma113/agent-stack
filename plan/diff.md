  AI-Stack 独有优势

  1. Memory 系统 - 完整的长期记忆:
    - Profile (用户偏好)
    - TaskState (任务状态)
    - Episodic (事件记忆)
    - Semantic (语义搜索)
    - Summary (自动摘要)
    - Token Budget 控制
  2. Knowledge 系统 - RAG 能力:
    - 代码索引 (AST 解析 + 向量化)
    - 文档爬取 (sitemap/网站递归)
    - BM25 + Vector 混合搜索
  3. 架构更模块化 - Monorepo 独立包:
    - @ai-stack/provider - 可单独使用
    - @ai-stack/memory - 可单独使用
    - @ai-stack/knowledge - 可单独使用

  OpenCode 独有优势

  1. 更多 LLM 提供商 - 75+ vs 4
  2. TUI/Desktop/IDE - 多种界面选择
  3. Plan/Build 模式 - 先预览后执行
  4. Undo/Redo - 变更回滚
  5. LSP 集成 - 代码智能 (跳转定义、引用查找)
  6. 分享功能 - /share 生成对话链接
  7. AGENTS.md 生成 - /init 自动分析项目
